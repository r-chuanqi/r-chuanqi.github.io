关于系统

系统概述

智能知识库学习考试系统 是一个基于 Hexo 静态网站生成器构建的企业级知识管理与在线考试平台。系统集成了文档管理、权限控制、在线测验等核心功能，为组织提供安全、高效的知识沉淀与学习考核解决方案。

核心功能模块

1. 用户登录鉴权

  • 双角色体系：支持管理员（admin）和普通用户（user）两种角色
  • 前端权限拦截：未登录用户无法访问任何受保护页面，自动跳转至登录页
  • 会话管理：基于 sessionStorage 的登录态存储，有效期 24 小时
  • 登录页面：卡片式居中布局，支持角色切换、密码显隐、错误震动反馈

2. 权限分级控制

  • 文章级权限：管理员可将整篇文章标记为”仅管理员可见”（admin_only: true），普通用户连文章入口都看不到
  • 段落级权限：使用 HTML 注释标记 <!-- admin-only-start --> 和 <!-- admin-only-end --> 包裹敏感段落，普通用户看到的是”此内容仅管理员可见”的提示
  • 自动拦截：普通用户直接访问管理员专属文章 URL 会被 302 重定向到首页

3. 知识文档管理

  • Markdown 编辑：所有文章使用 Markdown 格式编写，简单易用
  • 分类与标签：支持多级分类和灵活标签系统
  • 目录自动生成：文章标题层级自动生成目录（Table of Contents）
  • 代码高亮：支持多种编程语言的语法高亮显示
  • 搜索功能：支持本地全文搜索和第三方搜索服务

4. 在线考试测验

  • 答题模块：支持在文章中嵌入交互式测验题目
  • 自动评分：客观题自动判分并显示结果
  • 答案解析：测验完成后查看正确答案和解析

5. 页面功能特性

  • 响应式布局：适配桌面端、平板和手机端
  • 图片灯箱：点击图片放大查看，支持画廊模式
  • 代码复制：代码块一键复制到剪贴板
  • 回到顶部：页面滚动超过 500px 后显示火箭按钮
  • 访问量统计：集成不蒜子页面浏览量统计

使用说明

登录访问

  1. 访问系统域名，自动跳转到登录页面
  2. 选择角色（管理员/普通用户）
  3. 输入对应密码，点击登录
  4. 登录成功后进入系统首页

管理员操作

  • 所有普通用户可访问的内容
  • 查看标记为”仅管理员可见”的文档和段落
  • 添加新文章（本地操作后重新部署）

普通用户操作

  • 浏览公开的知识文档
  • 参与在线考试测验
  • 查看分类和标签导航

安全说明

  • 登录态存储在浏览器 sessionStorage 中，关闭标签页即失效
  • 管理员专属内容在 HTML 层面被完全移除，无法通过查看源码获取
  • 建议定期更换登录密码，保证系统安全
